# ระบบจัดเก็บค่าน้ำประปาหมู่บ้าน — Full Ready Version

## วิธีใช้

```bash
npm install
npm run dev
```

เปิด `http://localhost:3000`

## Login เริ่มต้น

- Username: `admin`
- Password: `1234`

## ฟีเจอร์

- Dashboard มือถือ
- Login / Logout / Session timeout 3 ชั่วโมง
- จัดการผู้ใช้น้ำ
- จดมิเตอร์รายเดือน
- รองรับมิเตอร์ถอยหลัง / มิเตอร์วนกลับ
- เฉพาะค่าบริการ / ตัดมิเตอร์
- รับชำระ / ยกเลิกชำระ
- รายงาน / ค้นหา / ข้อมูลย้อนหลัง
- ใบแจ้งหนี้ / ใบเสร็จ / พิมพ์ A4
- Export CSV / JSON
- Backup / Restore
- Admin Tools ตรวจและซ่อมข้อมูล
- ตั้งค่าค่าน้ำ รอบบิล รหัสผ่าน

## หมายเหตุ

เวอร์ชันนี้ใช้ `localStorage` เหมาะกับใช้งานเครื่องเดียว ถ้าต้องใช้หลายเครื่องพร้อมกัน ควรย้ายฐานข้อมูลไป Supabase

## อัปเดตค่าบริการเฉพาะราย

- ผู้ใช้น้ำแบบ `เฉพาะค่าบริการ` สามารถกำหนดค่าบริการเองรายคนได้
- ถ้าไม่กรอก ระบบจะใช้ค่าบริการกลางจากหน้า Settings
- ค่าบริการเฉพาะรายจะแสดงในหน้าจดมิเตอร์ รับชำระ รายงาน ใบแจ้งหนี้ และใบเสร็จ
